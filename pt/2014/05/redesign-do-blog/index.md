Acabei de remover várias features inúteis / bloated para esse blog. Por
exemplo, removi todas as categorias: agora só existem tags. Além do mais,
retirei vários widgets que ninguém utilizava. E, finalmente, mudei o tema!
Espero que gostem do novo formato.

A inspiração veio do Jekyll, que é uma blogging engine simples porém elegante.
