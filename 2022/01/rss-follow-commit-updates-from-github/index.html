RSS: Follow commit updates from Github

14 Jan, 2022 • 36 words

It is possible to track/follow commits of git repositories on GitHub via RSS: https://github.com/<user>/<repo>/commits.atom.

For example, my dotfiles: https://github.com/thiagowfx/dotfiles/commits.atom

I am using this to keep track of Miniflux commits from within Miniflux itself (a RSS webapp).
